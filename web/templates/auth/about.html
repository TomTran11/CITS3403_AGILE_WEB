About UniApp

UniApp is a web application designed to help university students connect with others who share similar interests, study goals, and backgrounds. It aims to make student life more engaging by simplifying the process of finding like-minded peers.

Overview

University environments can be overwhelming, especially for new students. Many struggle to meet people outside their classes or form meaningful connections. UniApp addresses this by allowing users to create a profile and match with others based on shared characteristics such as study units and spoken languages.

By focusing on relevant attributes, the platform promotes more meaningful interactions rather than random connections, helping users find study partners, collaborators, or friends.

Key Features

  • Secure user authentication (signup, login, logout)
  • Interactive profile creation with study units and languages
  • Dynamic language selection using AJAX (Fetch API)
  • Real-time input validation (JavaScript)
  • Server-side validation and CSRF protection
  • Clean and responsive user interface

Technology Stack

  • Backend: Python (Flask)
  • Frontend: HTML, CSS, JavaScript
  • Templating: Jinja2
  • Database: SQLite
  • AJAX: Fetch API for dynamic content

Security

  • Password hashing using Werkzeug
  • CSRF protection with Flask-WTF
  • Client-side and server-side validation

Purpose & Motivation

This project was developed to apply full-stack web development concepts in a practical environment. It demonstrates the integration of frontend and backend technologies, along with secure data handling and dynamic user interaction.

The application also reflects an interest in building solutions that improve student experience and foster meaningful social connections within university communities.

Future Improvements

  • Advanced matching system based on multiple attributes
  • User dashboard with profile editing features
  • Real-time messaging system (WebSocket integration)
  • Email verification and account security enhancements
  • Improved UI/UX and mobile responsiveness
